# JavaScript Full Stack Learning Project

This repository contains beginner-friendly JavaScript practice files created as part of a full stack development learning journey. It includes simple examples of core JavaScript concepts, DOM manipulation, loops, functions, conditional statements, and basic HTML/CSS demonstrations.

## Project Overview

The goal of this project is to build a strong foundation in front-end JavaScript by practicing:

- Variables and data types
- Arrays and objects
- Conditional statements
- Loops
- Functions and arrow functions
- DOM access and updates
- Basic HTML structure
- CSS box model concepts

## Files Included

- `script.js` - Basic JavaScript syntax, variables, arrays, and objects
- `conditional-statement.js` - `if`, `else if`, `switch`, and ternary operator examples
- `loops.js` - `for`, `for...of`, and `while` loop examples
- `function.js` - Function declaration, function expression, arrow function, and even number example
- `Dom.js` - Basic DOM selection and content update
- `BoxModel.html` - Simple page demonstrating the CSS box model
- `conditional.html` - HTML file for conditional statement practice
- `function.html` - HTML file for function practice
- `loops.html` - HTML file for loop practice
- `Dom.html` - HTML file for DOM practice

## How to Use

1. Clone this repository:

```bash
git clone https://github.com/ranjithkumar077/java-script-python-full-stack-development.git
```

2. Open the project folder in VS Code or any code editor.

3. Run the `.html` files in your browser to test the examples.

4. Open the browser console to view JavaScript output.

## Learning Purpose

This project is intended for practice and learning. Each file focuses on a specific JavaScript concept, making it easier to understand the basics step by step.
